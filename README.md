# Tweet This Page

This is the extension that allows to tweet the current page.

# Features

- Add `Tweet This page` context menu

# Build

## Requirements

- `node`
- `yarn` or `npm`

## Build the extension

```shell
# Build a release version
npm run build
```

The extension will be built in the following directories.

- `dist` : for Google Chrome
